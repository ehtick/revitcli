# RevitCli v6.0 Local BIMOps Workbench Gap Report

This v6.0 slice is a contract baseline for the Local BIMOps Workbench and the
Revit Model Operations Ledger. It now includes read-only standards validate,
issue preflight, dry-run issue package, read-only deliverables verify,
read-only ledger query, read-only ledger validate, read-only ledger stats,
read-only ledger timeline, append-only local ledger records, ledger replay preview, and read-only
workflow registry commands over local artifacts. The portable command spine also executes signed `journal verify`,
`history list` JSON/table outputs, and `rollback --dry-run` against synthetic
local fixtures. Supported command-spine paths include table summary and
Markdown detail parity checks derived from JSON payloads. It does not implement
a ledger database runtime, Revit write behavior beyond approved `set` writes,
bounded set replay apply, bounded receipt-backed export replay apply, and
bounded manifest-backed schedule batch-export replay apply, SaaS, MCP,
built-in LLM behavior, or
dashboard-central state.
