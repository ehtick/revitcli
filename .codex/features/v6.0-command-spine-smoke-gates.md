## Purpose

Gate the non-ledger v6.0 Local BIMOps command spine with dedicated portable
smoke docs and release/workbench checks.

This is not a new command surface, not a Revit write feature, not SaaS, not MCP,
not dashboard-central state, and not built-in LLM behavior.
