## Purpose

Make sheet release metadata safe enough for production pilots: verify sheet
state, plan issue code/date updates, apply with receipts, and rollback without
guessing. This plan must not start while the v5.0 RC gate is `NO-GO`.

## Commands

- `sheets verify`
- `sheets issue-meta`
- `sheets renumber`
- `sheets index`
- `plan show`
- `plan apply`
- `rollback`
- `journal verify`
